# Release package notes

This ZIP is a runtime install package for WooCommerce sites. It contains the PHP source, built admin assets and documentation required to install and run the plugin.

For WordPress.org or third-party source review, publish the matching development repository or submit an expanded source package that includes the original asset source files, build scripts, dependency manifests and reproducible build instructions for the files under `assets/build/`.

Do not claim the runtime ZIP alone is a complete reproducible source package for WordPress.org review unless the matching build source is supplied separately.
